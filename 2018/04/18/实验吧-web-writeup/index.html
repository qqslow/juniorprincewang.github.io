HTTP分割注入（HTTP Parameter Fragment）

基于错误回显的注入为
updatexml(1,concat(0x7e,version(),0x7e),1)，但是这条语句不能通过单一的参数传，两个参数过滤的关键字不同，
可以采用HTTP分割注入：username 未过滤 updatexml ，而 password 未过滤 ()， 中间的语句通过注释符过滤掉。
