kvm虚拟机磁盘扩容

By Max

Published 2018-04-13

Contents

kvm虚拟机的磁盘空间告急，怎么有效进行扩容。

kvm的磁盘格式分为：raw磁盘格式和qcow2磁盘格式，扩充的思路如下:

假如对raw格式的 centos6.6.img 直接扩展：

image: centos6.6.img
file format: raw
virtual size: 16G (17179869184 bytes)
disk size: 16G
image: centos6.6.img
file format: raw
virtual size: 26G (27917287424 bytes)
disk size: 16G

开启虚拟机，然后查看磁盘空间。
